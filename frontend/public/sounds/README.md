# Game sound effects

Add these MP3 files (short, under 1–2 seconds):

- **correct.mp3** — played when a card is placed correctly
- **wrong.mp3** — played when a card is placed incorrectly
- **victory.mp3** — played when the match ends with a winner (can be 1–2 seconds)
- **defeat.mp3** — short, light defeat sound when the player loses the match
- **tick.mp3** — short, soft arcade-style tick for the last 3 seconds of the turn timer

Keep volume moderate to avoid being loud or annoying.
